# Project 4 — Lending Library Single Page App

Browser-based single page application for the lending library system, connecting to the Project 3 REST API.

## What This Project Does
- Search books by keyword with pagination
- View book details
- Checkout books by patron ID
- Return books
- Real-time borrower list updates

## Tech Stack
- TypeScript
- DOM API
- Connects to Project 3 Express backend at https://localhost:2345

## How to Run
```bash
npm ci
npm run build
npm start
```

Open http://localhost:2346 in your browser.

## Requirements
- Project 3 server must be running at https://localhost:2345
